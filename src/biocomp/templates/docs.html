Documentation

BioComp is a web tool for estimating biomass composition using ultimate and chemical analysis data. Calculations are based on the biomass characterization method discussed in the Debiagi, Ranzi, et al. papers (see References section). Also, see the documentation for the Chemics biocomp function for implementation details of the characterization procedure.

The BioComp source code is available on GitHub. Submit a Pull Request if you would like to contribute to the project. Questions and other comments can be submitted on the GitHub Issues page. You can also contact Gavin Wiggins via Email or Mastodon.

Parameters

A description of the input parameters are given below. The ultimate and chemical analysis parameters refer to measured data for a particular biomass feedstock. The only required parameters for BioComp are carbon and hydrogen from ultimate analysis data. However, it is recommended to provide as much of the parameters as possible for best results.

Ultimate analysis

  • Carbon % - mass percent of carbon (C), dry ash-free basis
  • Hydrogen % - mass percent of hydrogen (H), dry ash-free basis
  • Hâ‚‚O % - mass percent of water, as received basis
  • Ash % - mass percent of ash, as received basis

Chemical analysis

  • Cellulose % - mass percent of cellulose (CELL), dry ash-free basis
  • Hemicellulose % - mass percent of hemicellulose (HEMI), dry ash-free basis
  • Lignin % - mass percent of total lignin, dry ash-free basis

Splitting parameters

  • Î± - molar ratio of cellulose and hemicellulose
  • Î² - molar ratio of lignin LIG-O and lignin LIG-C
  • Î³ - molar ratio of lignin LIG-H and lignin LIG-C
  • Î´ - molar ratio of lignins (LIG-H and LIG-C) and extractive TGL
  • Îµ - molar ratio of lignins (LIG-O and LIG-C) and extractive TANN

Optimize

  • If selected, optimized splitting parameters are determined using the given ultimate and chemical analysis values. The biomass composition is determined using the optimized splitting parameters.
  • If not selected, the biomass composition is calculated using the given or default splitting parameter values.

Results

The results will appear after pressing the Enter (Return) key or clicking the Submit button.

References

The references for estimating biomass composition are given below.

  1. Paulo Debiagi, Chiara Pecchi, Giancarlo Gentile, Alessio Frassoldati, Alberto Cuoci, Tiziano Faravelli, and Eliseo Ranzi. Extractives Extend the Applicability of Multistep Kinetic Scheme of Biomass Pyrolysis. Energy & Fuels, vol. 29, no. 10, pp. 6544-6555, 2015.
  2. Paulo Debiagi, Giancarlo Gentile, Alberto Cuoci, Alessio Frassoldati, Eliseo Ranzi, Tiziano Faravelli. A predictive model of biochar formation and characterization. Journal of Analytical and Applied Pyrolysis, vol. 134, pp. 326-335, 2018.
